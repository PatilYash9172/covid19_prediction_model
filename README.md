This project showcases an **end-to-end Data Analytics solution** developed to monitor and visualize the COVID-19 pandemic's impact across India. ## üéØ Key Achievements & Deliverables

This dashboard serves as a comprehensive portfolio piece, demonstrating expertise in the full data science lifecycle:

* **Engineered Scalable Dashboard:** **Developed and deployed** a highly interactive web dashboard using **Dash** and **Plotly** for dynamic user exploration and visualization.
* **Time-Series Modeling:** **Implemented** a time-series framework (`covid19_prediction_model.py`) to calculate and plot crucial trends, including **7-Day Moving Averages** and **Daily Growth Rates**.
